# /status - Update Project Status

Quick status.md update mid-session.

## Instructions

1. **Read current status.md** - Load `.claude/status.md`

2. **Identify changes needed**:
   - What's now complete that wasn't?
   - What's now in progress?
   - Any new blockers or issues?
   - Has the current phase changed?

3. **Edit status.md** - Make the updates directly

4. **Report changes** - Tell the user what you updated

## Output Format

```
## Status Updated

**Changes made to status.md**:
- [list of changes]

**Current state**:
- Phase: [X]
- In progress: [Y]
- Blockers: [Z or None]
```

## When to Use

- After completing a significant task
- When you hit a blocker
- When switching focus areas
- When user asks "where are we?"
